## When to Use Each Skill

### Document Creation

**docx skill** - Use when:
- Creating reports, memos, specifications, proposals
- Need professional formatting (TOC, headers, page numbers)
- Working with Word templates
- Extracting/organizing content from Word docs
- Inserting images or tables into documents

**pdf skill** - Use when:
- Reading or extracting from PDFs
- Merging/splitting PDF documents
- Adding watermarks or rotating pages
- Filling PDF forms
- OCR on scanned documents
- Creating searchable PDFs

**pptx skill** - Use when:
- Creating slide decks or presentations
- Reading/parsing existing presentations
- Building pitch decks or technical talks
- Combining multiple slide files
- Working with templates and layouts

**xlsx skill** - Use when:
- Analyzing tabular data
- Creating reports with charts and formulas
- Cleaning messy CSV/Excel files
- Building dashboards with data
- Converting between data formats

### Development & Design

**frontend-design skill** - Use when:
- Building web interfaces or components
- Creating landing pages or dashboards
- Developing React/HTML/CSS applications
- Need distinctive, production-grade UI
- Avoiding generic AI aesthetics is critical

**canvas-design skill** - Use when:
- Creating visual art, posters, or graphics
- Need original design work (not code)
- Building marketing materials
- Creating conference posters or infographics

### Product & Company

**product-self-knowledge skill** - Use when:
- Working with Claude API
- Questions about Claude Code setup/features
- Understanding rate limits, pricing, models
- Comparing Claude.ai plans (Pro/Team/Enterprise)
- Need current, verified Anthropic product info

**brand-guidelines skill** - Use when:
- Creating Anthropic-branded materials
- Need company color scheme and typography
- Building internal documentation
- Any artifact with Anthropic look-and-feel
